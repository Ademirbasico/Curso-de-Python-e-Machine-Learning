# 🐍 Primeiros Conceitos de Python

## 📌 Sobre

Este repositório contém um resumo dos primeiros conceitos de Python, baseado em aula introdutória.  
O objetivo é consolidar o aprendizado e servir como material de revisão.

---

## 🚀 O que é Python?

Python é uma linguagem simples e fácil de entender, muito usada para:

- Automação
- Desenvolvimento web
- Inteligência artificial
- Scripts em geral

Uma vantagem é que não precisa de muita estrutura pra começar.

---

## 🧱 Estrutura básica

Um programa em Python roda de cima para baixo.

Exemplo:

print("Meu primeiro programa")

---

## 💬 Comentários

Servem pra explicar o código (não são executados).

### Comentário simples:

# Isso é um comentário

### Comentário maior (docstring):

"""
Comentário em várias linhas
"""

---

## 📦 Variáveis

São usadas pra guardar valores.

nome = "Ademir"  
idade = 20  
altura = 1.70  

### 🧠 Importante:

- Não precisa declarar tipo
- Python define automaticamente
- Pode mudar o valor depois

---

## 🔢 Tipos de dados

### Inteiro (int)

x = 10  

### Decimal (float)

y = 10.5  

### Texto (str)

nome = "Python"  

### Booleano (bool)

ativo = True  

---

## 🔍 Descobrindo o tipo

print(type(x))

## ⌨️ Entrada de dados

nome = input("Digite seu nome: ")

⚠️ Sempre vem como texto (string)

---

## 🔄 Conversão de tipos

idade = int(input("Digite sua idade: "))  
altura = float(input("Digite sua altura: "))  

---

## ➕ Operadores

| Operador | Função |
|----------|--------|
| + | soma |
| - | subtração |
| * | multiplicação |
| / | divisão |
| // | divisão inteira |
| % | resto |
| ** | potência |

Exemplo:

a = 10  
b = 3  

print(a + b)  
print(a * b)  
print(a ** b)
